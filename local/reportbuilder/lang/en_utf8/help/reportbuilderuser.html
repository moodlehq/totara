Report Builder: Show by user

This criteria is enabled by selecting the 'Show records by user' checkbox. If selected, then the report will show different records depending on who the person viewing the report is, and their relationship to other users.

'Include records from a particular user' determines which records the user sees:

  • Set to 'A user's own records':
    The viewer of the report sees their own records.
  • Set to 'Records for user's direct reports':
    The viewer of the report sees records belonging to users who report to them. In other words, records appear for belonging to all users who's manager is the viewer of the report. A user's manager is set via the 'Position' tab in the user's profile.
  • Set to 'Both':
    The viewer sees their own records and their staff's records

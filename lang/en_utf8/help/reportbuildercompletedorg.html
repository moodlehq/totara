Report Builder: Show by completed organisation

This criteria is enabled by selecting the 'Show records completed in the user's organisation' checkbox. If selected, then the report will show different records depending on which part of the organisation the person viewing the report is in. This is determined by the settings in the 'Positions' tab in their user profile.

'Include records from child organisations' determines which records the user sees:

  • Set to 'no':
    The viewer of the report sees all records which were completed in the same organisation as the viewer is currently in.
  • Set to 'yes':
    The same as above, but the viewer of the report also sees records that were completed in organisations that lie below the viewer's organisation in the organisation hierarchy.

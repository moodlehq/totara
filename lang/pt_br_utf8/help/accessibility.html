Acessibilidade

Declaração de Acessibilidade

Esta instalação foi verificada em uma variedade de navegadores e com programas de leitura de tela.

O Moodle usa uma marcação semântica para ajudar os usuário de leitores de tela, dentre outros: instalação, páginas e cabeçalhos usam <h1>, <h2>, blocos de navegação são listas <ul> e assim por diante.

O Moodle segue o [Guia de Acessibilidade a Conteúdos na Rede 1.0] do Consórcio Word Wide Web, nível 1, a maioria nível 2 [ Conformidade Duplo A] e alguns nível 3.

Este guia procura atender à legislação local relativa ao acesso para pessoas com deficiências. No momento estamos trabalhando para melhorar a acessibilidade e usabilidade do Moodle. Existem ainda algumas tabelas usadas para a formatação, mas em geral a apresentação é separada do conteúdo (usando folhas de estilo, CSS).

Comentários e sugestões são bem-vindos, particularmente informar sobre qualquer conteúdo inacessível. Por favor [envie mensagem para o administrador].

Chaves de Acesso

No momento não implementamos chaves de acesso, mas elas estão previstas para uma próxima versão do Moodle.

Links

Há um link antes de cada bloco lateral que permite aos usuários de leitores de tela salterem o bloco. Note que este link fica oculto nos navegadores gráficos.

Muitos links tem atributos de título que o descrevem em mais detalhes, a menos que o texto do link já descreva completamente para onde aponta (por exemplo, o cabeçalho de um artigo). Se um link tem uma chave de acesso isto será anunciado no título do link.

Sempre que possível, os links são escritos para serem entendidos fora do contexto. Não há "pseudo-links" javascript.

Imagens

Todas as imagens usam o atributo alt para prover textos alternativos onde for apropriado. Imagens que são puramente decorativas têm o atributo alt vazio. Imagens que apresentam informações complexas, como os mapas, tem um atributo longdesc associando-as a um texto que descreve o contéudo da imagem.

Conformidade com Padrões

Este ambiente é construído usando XHTML válido para marcação e usa CSS para apresentação.
